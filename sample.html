Lispy - Syntax Highlighting for (Common) Lisp

Here's the old standby, Hello World!

(defun hello (who)
  (format t "Hello, ~a!~%" who))

Here's a more complicated example, a function that builds subsequent in-order sublists given a single input list (this crops up a lot in compiler writing, as it turns out):

(defun sublists (lst)
  (labels ((build-me-up (last args)
             (cond ((null args) nil)
                   (t (let ((new-last (append last (list (car args)))))
                        (cons new-last
                              (build-me-up new-last (cdr args))))))))
     (build-me-up nil lst)))
